This repository contains useful scripts for programmers.

cpdirs.sh is a script that copies two directores into a specified destination directory.
grepdir.sh is a script that looks for a pattern string inside all the files in a specified directory.
makemake.sh is a script that automatically generates a Makefile based on the source files, header files, and specified executable name.
uncomp.sh is a script that automatically uncompresses a file correctly based on its extension (tar or unzip).
